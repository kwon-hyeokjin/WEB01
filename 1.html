HTML이란 무엇인가?

Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web pages can be created and modified by using professional HTML editors.

However, for learning HTML we recommend a simple text editor like Notepad (PC) or TextEdit (Mac).

We believe using a simple text editor is a good way to learn HTML.
